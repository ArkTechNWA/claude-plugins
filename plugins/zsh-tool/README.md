# zsh-tool Plugin

This plugin provides the zsh-tool MCP server for Claude Code.

## Quick Install

```bash
pip install zsh-tool
```

Then add to your MCP config (`~/.claude/mcp_servers.json`):

```json
{
  "zsh-tool": {
    "command": "zsh-tool"
  }
}
```
